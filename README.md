**Neural Marker Analysis Macros for FIJI/ImageJ**

 This repository contains three specialized macros designed for the automated quantification of cFos, Perineuronal Nets (PNN), and Parvalbumin (PV) in brain tissue sections.

**OVERVIEW**

These macros automate the pre-processing, segmentation, and quantification of common neuronal markers, ensuring consistency across different experimental batches.
- **cFos** = Neuronal Activation Marker -> optimized for detecting small, high-contrast punctate signals in the nucleus
- **PNN** = Perineuronal Nets -> designed to capture the complex, honeycomb-like extracellular matrix structures
- **PV** = Parvalbumin-positive Interneurons -> specialized in isolating neuronal somas from dense axonal and dendritic fields

**QUICK START**
1. Open FIJI/ImageJ.
2. Drag and drop the desired .ijm file into the main bar.
3. Click Run on the script editor with your image open.

Feel free to use it!
